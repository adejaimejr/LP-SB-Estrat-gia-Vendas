## 🎨 FLUXO VERTICAL (lado esquerdo) — Melhorar visualmente

O fluxo com os círculos vazios e texto está muito simples. Precisa de mais presença visual:

### Melhorias:

1. **Círculos preenchidos** ao invés de vazios:
```css
.flow-circle {
  width: 12px;
  height: 12px;
  background: #C9A84C;
  border-radius: 50%;
  box-shadow: 0 0 10px rgba(201, 168, 76, 0.3);
}
```

2. **Linha conectora visível** entre os círculos:
```css
.flow-line {
  width: 2px;
  height: 100%;
  background: linear-gradient(to bottom, #C9A84C, rgba(201, 168, 76, 0.2));
  position: absolute;
  left: 5px; /* center of circle */
}
```

3. **Numeração dourada** antes de cada item:
```
① Origem
② Captura
③ Automação
④ Atendimento
⑤ Conversão
⑥ Análise
```
Usar números dourados `color: #C9A84C`, `font-weight: 700`, `font-size: 0.85rem` antes de cada label.

4. **Descrição curta** em cinza abaixo de cada etapa:
```
① Origem → "De onde vem o lead"
② Captura → "Landing page + formulário"
③ Automação → "n8n + webhooks + API"
④ Atendimento → "WhatsApp + CRM"
⑤ Conversão → "Script + follow-up"
⑥ Análise → "Dashboard + relatórios"
```
Essas descrições curtas ficam em `color: #888888`, `font-size: 0.8rem`, logo abaixo do nome da etapa.

5. **Animação de entrada:** Quando a seção entra na viewport, os itens do fluxo aparecem de cima pra baixo com stagger de 0.15s entre cada, e a linha dourada "desenha" de cima pra baixo simultaneamente.
